# N-Queens Explorer

- Built an interactive N-Queens Explorer using HTML, CSS, and JavaScript with real-time board rendering and dynamic animation controls.
- Implemented a recursive backtracking algorithm to solve up to 8 queens, visualizing valid placements and conflict detection.
- Responsive UI with adjustable speed settings and board generation, dynamically displaying all possible solutions for user-entered values.
